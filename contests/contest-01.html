Problem A — Locker Loop

CF ~800 Implementation

Jerry Chen is speed-walking the main-floor hallway at Marc Garneau Collegiate. The lockers are numbered 1 to L in a loop. Starting at locker S, he takes N moves. Each move is either + k (forward) or - k (backward). The hallway wraps around.

Input
  • First line: integers L, S, N
  • Next N lines: a character + or -, and integer k
Output
  • The final locker number
Constraints
  • 1 ≤ L ≤ 109
  • 1 ≤ S ≤ L
  • 1 ≤ N ≤ 200000
  • 0 ≤ k ≤ 109
Sample Input
10 3 4
+ 4
- 2
+ 10
- 5
Sample Output
10

Problem B — Cafeteria Pairing

CF ~1000 Greedy (two pointers)

At lunch, Michael Zhang and Bryan Yao are trying to seat students in pairs so the cafeteria stays survivable. There are N students; student i has noise level ai. A pair is valid if the sum of their noise levels is ≤ C. Each student can be used at most once. Find the maximum number of valid pairs.

Input
  • First line: integers N and C
  • Second line: N integers a1 ... aN
Output
  • Maximum number of pairs
Constraints
  • 1 ≤ N ≤ 200000
  • 1 ≤ ai ≤ 109
  • 1 ≤ C ≤ 2 × 109
Sample Input
6 10
2 3 7 8 4 5
Sample Output
2

Problem C — Holiday Show Badges

CF ~1100–1200 Ad-hoc validation

It’s the Holiday Show at MGCI, and the badge printer got… sabotaged. A badge code is a string S (length N) consisting of G, M, and ?. Each ? must be replaced with G or M.

A badge code is valid if:

  • It contains exactly N/2 G and N/2 M (so N must be even), and
  • For every prefix, the number of G is at least the number of M.

Determine if it’s possible to replace all ? to make S valid.

Input
  • One line: string S
Output
  • YES if possible, otherwise NO
Constraints
  • 1 ≤ N ≤ 200000
  • S contains only G, M, ?
Sample Input
G??M
Sample Output
YES

Problem E — Garneau Shortcut Pass

CF ~1500–1600 Dijkstra (2-state) USACO Silver max

Jerry Chen, Bryan Yao, and Michael Zhang are racing from the front entrance (node 1) to the gym (node N) along hallways modeled as a weighted undirected graph with M edges. Traversing an edge of weight w costs w time.

They have one shortcut pass that can be used on at most one edge to reduce its cost to floor(w/2). Compute the minimum possible travel time from node 1 to node N.

Input
  • First line: integers N, M
  • Next M lines: integers u, v, w
Output
  • Minimum possible travel time
Constraints
  • 2 ≤ N ≤ 200000
  • 1 ≤ M ≤ 200000
  • 1 ≤ u, v ≤ N, u ≠ v
  • 0 ≤ w ≤ 109
Sample Input
5 6
1 2 10
2 5 10
1 3 5
3 4 5
4 5 20
2 3 2
Sample Output
12

A — Locker Loop

Idea: sum net movement delta, wrap with modulo. Time: O(N).

import sys

def main():
    it = sys.stdin.buffer.readline
    L, S, N = map(int, it().split())
    delta = 0
    for _ in range(N):
        line = it().decode().strip().replace(" ", "")
        sign = line[0]
        k = int(line[1:])
        delta += k if sign == '+' else -k
    print(((S - 1 + delta) % L) + 1)

if __name__ == "__main__":
    main()
B — Cafeteria Pairing

Idea: sort + two pointers. If smallest+largest ≤ C, pair them. Time: O(N log N).

import sys

def main():
    data = list(map(int, sys.stdin.buffer.read().split()))
    N, C = data[0], data[1]
    a = sorted(data[2:])
    i, j = 0, N - 1
    ans = 0
    while i < j:
        if a[i] + a[j] <= C:
            ans += 1
            i += 1
            j -= 1
        else:
            j -= 1
    print(ans)

if __name__ == "__main__":
    main()
C — Holiday Show Badges

Idea: Ensure exact counts and prefix constraint. Fill enough ? with G first, then the rest with M, and validate prefixes. Time: O(N). 

import sys

def main():
    S = sys.stdin.readline().strip()
    n = len(S)
    if n % 2 == 1:
        print("NO")
        return

    target = n // 2
    fixedG = S.count('G')
    fixedM = S.count('M')
    if fixedG > target or fixedM > target:
        print("NO")
        return

    gNeed = target - fixedG
    out = list(S)
    for i in range(n):
        if out[i] == '?':
            if gNeed > 0:
                out[i] = 'G'
                gNeed -= 1
            else:
                out[i] = 'M'

    bal = 0
    for ch in out:
        bal += 1 if ch == 'G' else -1
        if bal < 0:
            print("NO")
            return

    print("YES" if bal == 0 else "NO")

if __name__ == "__main__":
    main()
D — Club Room Booking

Idea: min rooms = maximum overlap. Sort starts/ends and sweep. Time: O(N log N).

import sys

def main():
    data = list(map(int, sys.stdin.buffer.read().split()))
    N = data[0]
    starts, ends = [], []
    idx = 1
    for _ in range(N):
        s = data[idx]; e = data[idx+1]; idx += 2
        starts.append(s); ends.append(e)
    starts.sort()
    ends.sort()

    i = j = 0
    cur = best = 0
    while i < N:
        if starts[i] < ends[j]:
            cur += 1
            best = max(best, cur)
            i += 1
        else:
            cur -= 1
            j += 1
    print(best)

if __name__ == "__main__":
    main()
E — Garneau Shortcut Pass

Idea: Dijkstra with 2 states (pass unused/used). Time: O((N+M) log N).

import sys, heapq

def main():
    input = sys.stdin.buffer.readline
    N, M = map(int, input().split())
    g = [[] for _ in range(N+1)]
    for _ in range(M):
        u, v, w = map(int, input().split())
        g[u].append((v, w))
        g[v].append((u, w))

    INF = 10**30
    dist0 = [INF]*(N+1)
    dist1 = [INF]*(N+1)
    dist0[1] = 0
    pq = [(0, 1, 0)]  # (dist, node, used_pass)

    while pq:
        d, u, used = heapq.heappop(pq)
        if used == 0 and d != dist0[u]:
            continue
        if used == 1 and d != dist1[u]:
            continue

        for v, w in g[u]:
            nd = d + w
            if used == 0:
                if nd < dist0[v]:
                    dist0[v] = nd
                    heapq.heappush(pq, (nd, v, 0))
                nd2 = d + (w // 2)
                if nd2 < dist1[v]:
                    dist1[v] = nd2
                    heapq.heappush(pq, (nd2, v, 1))
            else:
                if nd < dist1[v]:
                    dist1[v] = nd
                    heapq.heappush(pq, (nd, v, 1))

    print(min(dist0[N], dist1[N]))

if __name__ == "__main__":
    main()
