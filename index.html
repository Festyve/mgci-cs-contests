About

These are unofficial practice contests made for the Marc Garneau CS Club community. Problems aim to feel like CCC-level tasks (especially Junior/Senior early–mid range), with a focus on ad-hoc reasoning and core techniques (two pointers, greedy, prefix sums, stacks).

Format

  • A–B: warmup / implementation
  • C–D: core CCC techniques
  • E: ad-hoc “hard” (USACO Silver / CF ~1500–1600), not super algo-heavy

Tip: If you want to “run” a contest, set a time limit, solve in order, and share submissions via Discord or Google Form.
